Detailed Description

This header file defines PrimalDualIPM class and other types/macros.

There exist several build configuration macros:

  • IPM_DISABLE_LOG disables logging function. This enables to cut dependency on std::ostream.
  • IPM_USE_FLOAT uses single floating point precision instead of double.
  • IPM_ENABLE_ASSERT enables assertion when generating error string. This is useful when using debugger because it breaks.
  • IPM_DECOMP_TYPE specifies which matrix decomposition is used for KKT matrix. FullPivLU, HouseholderQR, FullPivHouseholderQR and ColPivHouseholderQR are available. If not defined, JacobiSVD is used.

These are not defined by default.
